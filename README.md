# 六爻排盘离线网页原型

这是一个不依赖后端的单页网页原型，打开 `index.html` 即可使用。

## 在线访问

GitHub Pages: <https://sky-wanderer.github.io/liuyao-mobile/>

## 本地运行

直接双击 `index.html` 可以预览。若浏览器限制本地存储，可在当前目录运行：

```bash
python3 -m http.server 8000
```

然后打开 <http://localhost:8000>。

## 当前已实现

- 三枚硬币模拟摇卦
- 摇卦时显示无字周易铜钱翻转动画，结果文字落在铜钱旁边
- 六爻由下往上记录
- 6/7/8/9 及动爻变化
- 本卦、变卦名称查表，并列显示六爻
- 纳甲、六亲、世应、六神、伏神基础排盘
- 第一次摇卦时自动锁定起卦时间，并生成干支年、月、日、时与旬空
- 每爻显示旬空、月建、日辰对本卦爻的临、扶、生、克、合、冲、月破提示
- 摇满六爻后一键复制完整卦例，包含四柱、旬空、卦宫、六冲/六合、世应、伏神、动变状态
- 占问、备注和最近 30 条历史卦例保存在浏览器本地
- 移动端排盘风格布局

当前版本先采用常见纳甲与八宫世应表。用神、三合三会、驿马桃花、十二长生、墓库、反吟伏吟等断卦提示仍需按占问人工判断。
